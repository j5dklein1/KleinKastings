# Klein Kastings, Inc. - Aluminum Foundry Website

This is a modern, responsive website built with React, Vite, and Tailwind CSS for Klein Kastings, Inc.

## Key Features

- **Industrial Design Theme**: Uses a crisp "Precision Industrial" style with deep navy, metallic silver, and clean accent colors.
- **Video Hero Section**: Features a dynamic molten aluminum pouring video to immediately showcase the foundry environment.
- **Detailed Capabilities**: Dedicated sections for 3D Modeling/Design, Shell Process Casting, and CNC Machining/Finishing.
- **Industry Specifics**: Mentions Trucking, Furniture, and Custom Projects (like Golf Tee Markers).
- **Contact Information**: Easy-to-find phone, email, and address with map integration.
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices.

## Project Structure

- `src/pages/Home`: Main landing page with hero, services overview, and industry highlights.
- `src/pages/Capabilities`: Detailed breakdown of manufacturing processes.
- `src/pages/Contact`: Contact information and location map.
- `src/components`: Reusable UI components (Layout, Navbar, Footer, ScrollToTop).
- `src/routes`: Route configuration with lazy loading for performance.

## Tech Stack

- React 18+
- Vite
- Tailwind CSS
- Lucide React Icons
- React Router DOM

## Development

Run `pnpm run dev` to start the development server.
Run `pnpm run build` to create a production build.
